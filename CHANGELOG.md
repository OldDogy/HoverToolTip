# Changelog

All notable HoverToolTip changes are listed here.

## 0.1.0 - 2026-06-02

Initial standalone release.

- Added standalone GameTooltip styling for World of Warcraft Retail interface `120005`.
- Added configurable backdrop alpha, text alpha, scale, status bar visibility, font sizes, and font outlines.
- Added open-world player, NPC, quest, and object tooltip styling.
- Added optional unit info positioning above the unit name.
- Added safe restricted-instance handling for dungeons, raids, scenarios, and Delves.
- Added optional compact secret instance tooltip styling that avoids reading secret tooltip text.
- Added support for Blizzard unit frames and ElvUI unit frames.
- Added full-details modifier behavior and keybinding support.
- Added resizable in-game options window.
- Added named profiles with per-character profile selection.
- Added profile import/export.
- Added Minimal, Clean, and Full Info presets.
- Added tabbed options window sections for Setup, General, Text, and Lines.
- Added debug snapshot and trace commands for troubleshooting.
